C 语言基本语法
Aidan 努力的小白
 2021-09-17 17:26:24  
  • 学习 
 4k 字  15 分钟

当时准备 PAT 竞赛的时候临时准备的基础语法，因为这种计算机能力的考试多是使用 C/C++ 来参赛，所以 C 的学习还是有必要的，况且后续许多语言的基本语法也都大同小异，毕竟语言只是工具
当时看的视频是浙大翁恺，讲的非常好，这里只是简单的笔记，做个记录

简单运算

变量

  • 变量是保存数据的地方；
  • 定义形式：< 类型名称 >< 变量名称 >;
  • 其中变量的名字叫做标识符，是由字母数字下划线组成（数字不能放在开头），关键字不能做标识符
    int 型数据的范围是 −231-2^{31}−231 ~ 231−12^{31}-1231−1 之间，可记为绝对值小于10910^9109
    long long 型数据记录超过 int 型数据时后面要加上 LL：long long bignum=1234567894567LL;
    在整形数据前加上 unsigned 表示无符号型（即全是正数）

赋值

  • “=” 作为赋值符号（作为动作是动态的），程序中的相等为 “==”（作为关系是静态的）
  • C 语言不要求所有的变量必须被初始化，但是在第一次参与运算之前必须进行初始化（赋值）
  • 初始化形式：< 类型名称 >< 变量名称 > = < 初始值 >;
    C99 允许在任何地方进行变量的定义

常量

直接在程序中写的常数称为直接量，而将常数普遍定义为常量，这样做的好处是更容易理解常量数字存在的意义、便于修改
常量的定义：const int AMOUNT=100;
const 必须赋初值，表示这个变量不能作修改，若要用户对常量进行输入，需要用一个变量进行中间转换

浮点数

浮点型数据中 double 的精度要比 float 高，统一使用 double 即可

浮点数和整数放到一起运算时，C 会将整数转换为浮点，然后进行浮点数的运算

字符型

char 型字符统一使用 ASCII 码进行存储
可以使用 char=getchar(); 的方式进行字符输入

字符串

字符数组的读入无需取址

转义字符：’\0’ 代表的是空字符 NULL，ASCII 码值为 0，不是空格的意思

运算符

运算表达式由算子和运算符组成
其中算子就是变量、常量和常数；运算符为：+、-、*、/、%
运算符的优先级：正负级→乘除 / 取余→加减→赋值运算符

复合赋值

将算数运算符和赋值运算符合并使用：total += 5; == total=total+5;
++ 和 – 放到变量的前或后进行增减赋值

判断

关系运算符

计算两个值之间的关系叫做关系运算，使用的运算符号叫做关系运算符
关系运算符的优先级要低于算术运算符，但是高于赋值运算符

if

Form：

if（表达式）{
语句；
}else if（表达式）{
……
}

Tips：在所有的 if…else 后跟上大括号，便于嵌套判断的读写
级联 if…else if 用于分支判断，再多分支程序中，尽量使用单一出口

switch

Form：
switch（表达式）{
case 常量：
语句；
break；
defaul：
语句；
……
}
switch 语句的表达式必须是整型的，可以是常数，也可以是常数计算的表达式：case 1+1: == case 2:
在 C99 中可以使用常量名进行分支选择的判断：
如我们预先定义 const int MAX=1; ，那么 case MAX:== case 1:
case 只是分支的开始，switch 并不是对计算进行分割，如果 case 中语句完成后没有 break，那么之后的 case 还将继续执行

循环

循环有 while 循环、do…while 循环、for 循环

(do)while

Form：
while（循环条件）{
< 循环体语句 >;
}
do{
< 循环体语句 >;
}while（循环条件）;
区别： do…while 循环是先进入循环再判断条件，至少会循环一次；
while 循环是先判断条件再进入循环

Log 计算

输入对数和底数，求出指数

循环读入

while… EOF 格式进行数据的循环输入，也就是说当输入的值不是 EOF 时，一直进行文件的读入

C 的随机数

头文件导入：#include <stdlib.h>; #include <time.h>
语句实现：srand(time(0)); int number = rand() % 100 + 1;
for 循环
Form：
for（循环初始条件；循环继续条件；每轮变化操作）{
< 循环体语句 >;
}
变化操作必须对条件赋值：可以写成 i+=2 或者 i++ 但不能写成 i+2
直接在 for 循环中定义条件只有在 C99 才能进行
循环的控制

  • break： 跳出当前循环
  • continue： 跳过这一轮循环剩下的语句进入下一轮循环
    嵌套循环的跳出操作：
  1. break 接力：设置一个变量，当第一次 break 的时候将变量设置为 true，在更外一层的循环中判断变量的值决定是否继续跳出
  2. goto：使用 goto 语句指向出口，运行到 goto 语句后直接跳出到循环外的出口但是这种方式易破坏程序的结构性，尽量只在多重循环嵌套中使用

数据类型

C99 提供了 long long; long double; bool
sizeof() 加上变量名可以表现当前变量在内存中占用几个字节
二进制中负数的存储是使用补码的形式：-a 存储为 0-a，实际上是 2n−a2^n-a2n−a，其中 n 是是这种类型的位数

8 进制和 16 进制

8 进制在 C 语言中用 0 开头，16 进制用 0x 开头

浮点数

科学计数法输出：%e 或 %E

浮点型也可以用科学计数法进行存储 double f = 1E-10;
char 型： char 型既可以作为字符也可以作为整数，但尽量只作为字符使用，字符可配合 ASCII 码使用

类型转换

自动转换

当运算符两边字符不同会自动转换为较大范围的字符；如整数和浮点一起运算时，整数转换为浮点数

  • char → short → int → long
  • int → float → double
    输入输出时的转换：
  • 在进行 printf 时，小于 int 的整型会自动转为 int 型 (%d) 来进行输出，float 也会转换为 double 型进行输出
  • 在 scanf 时，则必须对类型指示正确，如 short 型整数，必须写成 scanf("%hd", &n);

强制转换

一般是将一个较大范围的字符转换为较小范围
Form：（类型）值 n=(int)10.2; 强制类型转换的优先级比运算要高

bool 型

包含在头文件 <stdbool.h>，0 为 false，非零为 true
printf 只能输出 1 或者 0，并没有对 true 或 false 的逻辑类型输出

逻辑运算符

短路： 逻辑运算的 "&&"、"||" 是自左向右的，也就是当左边成立，如 && 运算时左边已经为 false，那左边就不会运算
三目运算
Form：flag ? true : false

逗号运算

在 for 循环用作多条件的分割
赋值中会直接返回逗号之后的结果

函数

函数的定义

当函数的参数是数组时，无需定义长度，二维数组需要对第二维定义长度，但这里的数组在函数中的变化就是数组本身的变化，而不是局部变量

函数的调用

调用 Form：函数名（参数表）; 用以获取相应参数生成的函数返回值或者使全局变量发生改变
没有参数也要加上 "()"，否则便没法进行调用
没有返回值的函数使用 void 型进行定义

函数的先后关系

C 语言中的编译时自上而下的，所以需要在 main 方法中调用的函数没有定义在 main 之前则会编译报错
解决： 可以将函数头声明在开头，但不进行函数体的定义，在 main 之后进行函数体的定义，声明和定义的函数头必须一致，但是在声明中可以不定义参数名，仅声明参数的数据类型

参数的传递

参数传递的时候如果类型不一致会进行自动类型转换
调用函数是的参数如果是变量，那么函数接收到的仅仅是变量存储的值，而不是变量本身，所以没办法写出一个真正的 swap() 函数（可以用指针）

数组

数组的定义

Form:< 类型 > 变量名称 [元素数量];

数组范围都是从 0 开始，如大小为 100 的数组，那么下标为 0~99；
数组越界在 C 语言的编译器中是不会进行检查的

数组的初始化

简单初始零： 正常初始化数组为 0 需要在循环中为数组的每一个元素进行赋值，但在 C99 中可以采用 int num[10]={0}; 的方式来为数组的所有元素初始化为 0，这行代码的定义其实仅仅是将 num[0] 这个元素赋值为 0，但根据其语言特性，会为数组中其他没有指定值的数组元素赋值为 0
定位初始化： 在 C99 中可以在初始化代码块中直接指定下标进行赋值 int num[10] = {[2] = 3, 4}; 但是在 g++ 编译器中会报错，所以无需使用
数组大小：sizeof() 方法可以求出该变量在内存中占用的字节数，也就是当初始化完成的一个数组，那么它数组的字节数除单个元素的字节就是数组的大小 sizeof(num) / sizeof(num[0])
数组的复制： 不使用函数的情况下，只能对每一个元素进行单个赋值，而不能直接用数组名进行赋值

二维数组

其中列号是必须指定的，但是行号可以由编译器来数
如果二维数组或多维数组的空间过大，可写在 main() 函数之外，申请静态空间

指针

& 和 * 运算符
取址符 &：取变量的地址，它的运算对象必须是变量，在 scanf 函数中对变量进行输入，其实就是取地址后对该地址的存储对象进行填充
取值符 *：与指针搭配使用，用来获取当前指针指向的地址的变量的值

创建指针
一个用来保存地址的变量，如：int *p;
示例： 这时定义一个变量 i，如果 i 的的地址为一个十六进制数 0x2000，那么将 i 的地址取出来放到 p 中 int *p=&i; 就代表 p 此时存储的就是地址 2000
数组可以直接指向指针 int a[]; int *p=a;
如果在函数中定义的参数为指针变量，如 void f(int *p); 那么就说明是在调用的时候调用某个变量的地址；
需要给予的参数也就必须是一个地址，如 int i=f(&i); 这句话的内容就是将地址赋给值
在函数里面可以通过这个指针访问到外面的 i

指针的应用

指针版 swap 函数

指针没得到任何地址的时候，不要进行任何操作

指针与数组
指针指向的并不是一个单一的地址，如果是 char 那么确实是一个，因为 sizeof(char)==1，如果是 int 型，那么指向的将是四个连续的地址，sizeof(int)==4
指针指向的不是一片连续分配的空间，那么加 1 也是毫无意义的
指针之间的互相相减，也是根据类型来：一个指向 int 型数组的头 0，一个指向尾 5，那么尾减头得到的将是 5 而不是其中的内存地址 4*5=20

*p++
取出 p 当前所指向的那个数据之后再将 p 移动到下一个；

  • 的优先级低于 ++

指针类型
指针是没有类型的，但是如果指针指向不同的数据类型，指针之间的值也是不能随意调换的
类型转换： 可以采用（数据类型或 void) 的方式进行转换，但没有实际意义，只是让阅读代码的人用不同的眼光看待这个变量

指针的作用

  • 需要传入较大的数作为参数
  • 传入数组进行数组操作时
  • 函数的返回值不止一个结果时
  • 需要用函数来修改不止一个变量时
  • 动态申请的内存

结构体
结构体使用构造函数初始化，也可以创建多个构造函数，给每个构造函数定义不同的形参列表，以达到多形式初始化
